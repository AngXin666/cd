# 请假管理系统使用指南

## 📱 司机端使用指南

### 1. 提交请假申请

#### 方式一：直接提交
1. 进入"请假管理"页面
2. 点击"申请请假"按钮
3. 填写申请信息：
   - 选择请假类型（病假/事假/年假/其他）
   - 选择开始日期
   - 选择结束日期
   - 系统会自动计算并显示请假天数
   - 填写请假事由
4. 点击"提交申请"按钮
5. 等待管理员审批

#### 方式二：保存草稿后提交
1. 进入"请假管理"页面
2. 点击"申请请假"按钮
3. 填写部分信息
4. 点击"保存草稿"按钮（紫色）
5. 草稿保存成功，可以随时回来继续填写

### 2. 管理草稿

#### 查看草稿
1. 进入"请假管理"页面
2. 点击"草稿箱"标签
3. 查看所有保存的草稿（包括请假和离职申请）
4. 草稿卡片显示：
   - 类型标签（请假/离职）
   - 详细信息
   - 操作按钮

#### 编辑草稿
1. 在草稿箱中找到要编辑的草稿
2. 点击"编辑"按钮
3. 修改申请信息
4. 可以选择：
   - 点击"保存草稿"更新草稿
   - 点击"提交申请"直接提交

#### 提交草稿
1. 在草稿箱中找到要提交的草稿
2. 点击"提交"按钮
3. 确认信息无误后提交
4. 草稿转为正式申请，进入审批流程

#### 删除草稿
1. 在草稿箱中找到要删除的草稿
2. 点击"删除"按钮
3. 确认删除操作
4. 草稿被永久删除

### 3. 查看申请状态

#### 查看待审批申请
1. 进入"请假管理"页面
2. 默认显示"待审批"标签
3. 查看所有待审批的申请
4. 显示信息：
   - 请假类型
   - 请假时间
   - 请假天数
   - 审批状态

#### 查看已审批申请
1. 进入"请假管理"页面
2. 点击"已审批"标签
3. 查看所有已审批的申请
4. 显示信息：
   - 请假类型
   - 请假时间
   - 审批结果（通过/拒绝）
   - 审批意见

### 4. 提交离职申请

#### 直接提交
1. 进入"请假管理"页面
2. 点击"申请离职"按钮
3. 填写申请信息：
   - 选择期望离职日期
   - 填写离职原因
4. 点击"提交申请"按钮
5. 等待管理员审批

#### 保存草稿
1. 进入"请假管理"页面
2. 点击"申请离职"按钮
3. 填写部分信息
4. 点击"保存草稿"按钮
5. 草稿保存成功，可以随时回来继续填写

## 👨‍💼 管理员端使用指南

### 1. 仓库管理员审批权限

仓库管理员只能审批自己管辖仓库的申请：
- 可以查看自己仓库司机的请假申请
- 可以审批（通过/拒绝）自己仓库的申请
- 无法查看其他仓库的申请

### 2. 超级管理员审批权限

超级管理员只能审批未分配管理员的仓库申请：
- 当仓库未分配管理员时，超级管理员可以审批该仓库的申请
- 当仓库已分配管理员后，超级管理员自动失去该仓库的审批权限
- 这样确保了权限的清晰分层，避免权限冲突

### 3. 审批流程

1. 进入"请假审批"或"离职审批"页面
2. 查看待审批的申请列表
3. 点击申请卡片查看详情
4. 选择审批操作：
   - 点击"通过"按钮批准申请
   - 点击"拒绝"按钮拒绝申请
   - 可以填写审批意见
5. 确认审批操作
6. 申请状态更新，司机会看到审批结果

## 💡 使用技巧

### 1. 草稿功能的最佳实践
- 不确定请假时间时，先保存草稿
- 需要查询相关信息时，先保存草稿
- 避免重复填写，提高效率
- 定期清理不需要的草稿

### 2. 天数计算说明
- 系统自动计算请假天数
- 计算公式：结束日期 - 开始日期 + 1
- 包含起始日期和结束日期
- 例如：1月1日到1月3日 = 3天

### 3. 审批权限说明
- 司机只能看到自己的申请
- 仓库管理员只能审批自己仓库的申请
- 超级管理员只能审批未分配管理员的仓库申请
- 权限在数据库层面强制执行，确保数据安全

### 4. 常见问题

**Q: 草稿会被管理员看到吗？**
A: 不会。草稿只有创建者可以看到，只有提交后才会进入审批流程。

**Q: 可以修改已提交的申请吗？**
A: 不可以。已提交的申请无法修改，如需修改请联系管理员撤回后重新提交。

**Q: 为什么我看不到某个仓库的申请？**
A: 因为权限限制。仓库管理员只能看到自己仓库的申请，超级管理员只能看到未分配管理员的仓库申请。

**Q: 草稿会自动过期吗？**
A: 不会。草稿会一直保存，直到您提交或删除。

**Q: 可以同时保存多个草稿吗？**
A: 可以。您可以保存多个请假申请草稿和离职申请草稿。
