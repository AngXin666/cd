## 创建租户

点击"创建租户"按钮，填写以下信息：

### 必填信息
- **公司名称**：租户的公司名称
- **老板姓名**：租户老板的姓名
- **老板手机号**：租户老板的登录手机号
- **老板密码**：租户老板的登录密码

### 可选信息
- 联系人姓名
- 联系电话
- 联系邮箱
- 老板邮箱
- 租期到期时间

### 自动化流程
点击"创建"后，系统会自动：
1. 生成租户代码（如：tenant-001）
2. 创建租户记录
3. 创建独立的数据库 Schema
4. 创建所有业务表结构
5. 创建老板账号
6. 在租户数据库中创建老板 profile

整个过程完全自动化，无需手动配置！
