# 快速测试指南

## 🎯 测试目标
验证不同角色登录后能够正确跳转到对应的工作台。

---

## 📱 测试步骤

### 测试 1: 司机账号
```
1. 打开应用
2. 输入账号: admin1
3. 输入密码: 123456
4. 点击登录

✅ 预期结果:
- 显示"登录成功"提示
- 自动跳转到司机工作台
- 显示"你好，测试司机"
- 显示司机功能菜单（打卡、考勤、计件等）
```

### 测试 2: 管理员账号
```
1. 退出当前账号
2. 输入账号: admin2
3. 输入密码: 123456
4. 点击登录

✅ 预期结果:
- 显示"登录成功"提示
- 自动跳转到管理员工作台
- 显示"你好，普通管理员"
- 显示管理员功能菜单（数据汇总、计件报表、请假审批等）
- 可以切换查看东区和西区仓库
```

### 测试 3: 超级管理员账号
```
1. 退出当前账号
2. 输入账号: admin
3. 输入密码: 123456
4. 点击登录

✅ 预期结果:
- 显示"登录成功"提示
- 自动跳转到超级管理员工作台
- 显示"你好，超级管理员"
- 显示超级管理员功能菜单（仓库管理、用户管理、权限配置等）
- 可以切换查看所有仓库
```

## ✅ 检查清单

### 登录流程
- [ ] 输入账号密码正常
- [ ] 点击登录按钮有响应
- [ ] 显示"登录成功"提示
- [ ] 自动跳转到工作台

### 页面显示
- [ ] 用户名称显示正确
- [ ] 仓库信息显示正确
- [ ] 功能菜单完整显示
- [ ] 底部导航栏正常显示

### 权限验证
- [ ] 司机只能看到司机功能
- [ ] 管理员只能看到分配的仓库
- [ ] 超级管理员可以看到所有功能

---

## 🐛 问题排查

### 登录失败
1. 检查账号密码是否正确
2. 查看浏览器控制台错误信息
3. 验证数据库连接状态

### 跳转错误
1. 检查用户角色是否正确
2. 查看控制台是否有跳转错误
3. 验证路由配置

### 页面空白
1. 检查网络请求是否成功
2. 查看控制台错误信息
3. 验证用户权限
