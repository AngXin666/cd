## 4. 文档查找指南

### 4.1 按功能查找

如果您需要查找特定功能的文档，可以通过以下步骤：

1. 确定功能所属的模块（如仓库管理、车辆管理等）
2. 查看功能模块文档部分，找到对应的模块
3. 查看模块下的文档列表，找到需要的文档

### 4.2 按文档类型查找

如果您需要查找特定类型的文档，可以通过以下步骤：

1. 确定文档类型（如系统核心、权限系统、功能模块等）
2. 查看对应的文档部分，找到需要的文档类型
3. 查看文档列表，找到需要的文档

### 4.3 按关键词查找

如果您需要按关键词查找文档，可以使用文档工具的搜索功能，输入关键词进行搜索。
