# 超级管理员请假审批功能 - 快速使用指南

## 一、功能入口

**访问路径：** 超级管理员后台 → 请假审批

## 二、界面概览

### 页面顶部
- **统计数据**：显示请假和离职申请的总数、待审批、已通过、已驳回数量
- **快速搜索**：输入司机姓名、仓库或事由进行搜索
- **状态筛选**：快速切换查看全部、待审批、已通过、已驳回的申请

### 高级筛选（点击展开）
- 请假类型：病假、事假、年假、其他
- 所属仓库：选择特定仓库
- 当事人（司机）：选择特定司机
- 审批人：选择特定审批人或查看未审批的申请
- 时间范围：设置开始和结束日期

### 申请列表
- **请假申请**和**离职申请**两个标签切换
- 每条申请显示完整的信息和审批历史

## 三、常用操作

### 3.1 查看待审批申请

**步骤：**
1. 点击"待审批"状态按钮
2. 查看所有待审批的申请列表

**快捷方式：**
- 页面默认显示所有申请
- 顶部统计卡片显示待审批数量

### 3.2 审批申请

**通过申请：**
1. 找到需要审批的申请
2. 点击"通过"按钮
3. 在确认对话框中点击"确定"
4. 系统显示"已通过"提示

**驳回申请：**
1. 找到需要驳回的申请
2. 点击"驳回"按钮
3. 在确认对话框中点击"确定"
4. 系统显示"已驳回"提示

**注意：**
- 只有"待审批"状态的申请才显示操作按钮
- 审批后会自动记录审批人和审批时间

### 3.3 搜索特定申请

**快速搜索：**
1. 在搜索框输入关键字（司机姓名、仓库名称或事由）
2. 系统实时显示匹配结果
3. 点击搜索框右侧的"×"清除搜索

**示例：**
- 搜索"张三"：显示所有司机姓名包含"张三"的申请
- 搜索"东区"：显示所有东区仓库的申请
- 搜索"感冒"：显示所有事由包含"感冒"的申请

### 3.4 按条件筛选

**单条件筛选：**
1. 点击"高级筛选"展开筛选选项
2. 选择需要的筛选条件
3. 系统自动更新列表

**多条件组合筛选：**
1. 展开高级筛选
2. 同时设置多个筛选条件
3. 系统显示同时满足所有条件的申请

**示例场景：**

**场景1：查看东区仓库的所有病假申请**
1. 展开高级筛选
2. 请假类型选择"病假"
3. 所属仓库选择"东区仓库"
4. 查看筛选结果

**场景2：查看某个司机的所有请假记录**
1. 展开高级筛选
2. 当事人（司机）选择目标司机
3. 查看该司机的所有请假记录

**场景3：查看本月的所有申请**
1. 展开高级筛选
2. 设置开始日期为本月1号
3. 设置结束日期为本月最后一天
4. 查看本月的所有申请

**场景4：查看未审批的申请**
1. 点击"待审批"状态按钮
2. 或在高级筛选中选择审批人为"未审批"

### 3.5 清空筛选条件

**方法1：点击清空按钮**
1. 展开高级筛选
2. 点击"清空所有筛选条件"按钮
3. 所有筛选条件重置为默认值

**方法2：手动重置**
- 点击"全部"状态按钮
- 清空搜索框
- 重新选择筛选条件

## 四、申请信息说明

### 请假申请卡片

**标题栏：**
- 请假类型（病假/事假/年假/其他）
- 审批状态（待审批/已通过/已驳回）

**当事人信息：**（蓝色背景）
- 司机姓名
- 所属仓库

**请假详情：**
- 请假时间：开始日期 至 结束日期
- 请假天数：自动计算
- 请假事由：详细说明
- 申请时间：申请创建时间

**审批历史：**（灰色背景，仅已审批申请显示）
- 审批人姓名
- 审批时间
- 审批意见

### 离职申请卡片

**标题栏：**
- 离职申请标识
- 审批状态

**当事人信息：**（橙色背景）
- 司机姓名
- 所属仓库

**离职详情：**
- 预计离职日期
- 离职原因
- 申请时间

**审批历史：**（灰色背景，仅已审批申请显示）
- 审批人姓名
- 审批时间
- 审批意见

## 五、状态说明

### 审批状态

| 状态 | 颜色 | 说明 |
|------|------|------|
| 待审批 | 橙色 | 等待审批的申请，可以进行审批操作 |
| 已通过 | 绿色 | 已经批准的申请，不可再次审批 |
| 已驳回 | 红色 | 已经驳回的申请，不可再次审批 |

### 请假类型

| 类型 | 说明 |
|------|------|
| 病假 | 因病需要休息的请假 |
| 事假 | 因私事需要请假 |
| 年假 | 年度带薪休假 |
| 其他 | 其他类型的请假 |

## 六、常见问题

### Q1：如何查看某个司机的所有请假记录？
**A：** 展开高级筛选 → 在"当事人（司机）"下拉框中选择该司机 → 查看筛选结果

### Q2：如何查看未审批的申请？
**A：** 点击"待审批"状态按钮，或在高级筛选中选择审批人为"未审批"

### Q3：如何查看某个时间段的申请？
**A：** 展开高级筛选 → 设置开始日期和结束日期 → 查看筛选结果

### Q4：如何同时按多个条件筛选？
**A：** 展开高级筛选 → 同时设置多个筛选条件 → 系统自动显示同时满足所有条件的申请

### Q5：审批后可以撤销吗？
**A：** 当前版本不支持撤销审批，请谨慎操作。审批前会有确认对话框。

### Q6：如何快速找到某个司机的申请？
**A：** 在搜索框直接输入司机姓名，系统会实时显示匹配结果

### Q7：筛选条件太多了，如何快速重置？
**A：** 展开高级筛选 → 点击"清空所有筛选条件"按钮

### Q8：为什么有些申请没有审批历史？
**A：** 只有已审批（已通过或已驳回）的申请才会显示审批历史，待审批的申请不显示

### Q9：请假天数是如何计算的？
**A：** 系统自动计算从开始日期到结束日期的天数（包含首尾两天）

### Q10：如何区分请假申请和离职申请？
**A：** 通过页面中部的标签切换，或通过卡片的图标和颜色区分（请假-蓝色，离职-橙色）

## 七、操作技巧

### 技巧1：快速审批待审批申请
1. 点击"待审批"状态按钮
2. 从上到下依次审批
3. 审批后自动刷新列表

### 技巧2：批量查看某类申请
1. 使用状态筛选 + 请假类型筛选
2. 例如：查看所有待审批的病假申请

### 技巧3：追踪审批进度
1. 使用审批人筛选
2. 查看自己已审批的申请
3. 或查看其他审批人的审批情况

### 技巧4：定期检查
1. 每天查看待审批数量（顶部统计卡片）
2. 及时处理待审批申请
3. 避免申请积压

### 技巧5：使用搜索快速定位
1. 记住司机姓名或仓库名称
2. 直接搜索，比逐个筛选更快
3. 适合查找特定申请

## 八、注意事项

### 审批操作
- ⚠️ 审批操作不可撤销，请仔细核对信息后再操作
- ⚠️ 审批前会弹出确认对话框，请认真阅读
- ⚠️ 审批后会自动记录审批人和审批时间

### 筛选功能
- ℹ️ 所有筛选条件可以组合使用
- ℹ️ 筛选是实时的，无需点击"确定"按钮
- ℹ️ 清空筛选条件会重置所有筛选

### 数据刷新
- ℹ️ 审批后会自动刷新数据
- ℹ️ 页面显示时会自动加载最新数据
- ℹ️ 如需手动刷新，可以切换标签或重新进入页面

### 信息查看
- ℹ️ 当事人信息用彩色背景高亮显示
- ℹ️ 审批历史仅在已审批申请中显示
- ℹ️ 操作按钮仅在待审批申请中显示

## 九、快捷操作流程

### 流程1：日常审批流程
```
进入页面
  ↓
查看待审批数量
  ↓
点击"待审批"状态
  ↓
查看申请详情
  ↓
点击"通过"或"驳回"
  ↓
确认操作
  ↓
完成审批
```

### 流程2：查找特定申请
```
进入页面
  ↓
输入搜索关键字
  ↓
查看搜索结果
  ↓
找到目标申请
  ↓
查看详情或审批
```

### 流程3：按条件筛选
```
进入页面
  ↓
展开高级筛选
  ↓
设置筛选条件
  ↓
查看筛选结果
  ↓
清空筛选（可选）
```
