# 司机端快捷功能说明

## 功能概览

司机端工作台提供4个核心快捷功能，采用2×2网格布局，方便司机快速访问常用功能。

## 快捷功能列表

### 1. 计件录入 📝
**图标**：蓝色剪贴板编辑图标  
**功能**：快速录入计件记录  
**跳转页面**：`/pages/driver/piece-work-entry/index`

**使用场景**：
- 完成一批货物配送后，立即录入计件数据
- 记录配送件数、上楼费、分拣费等信息
- 支持编辑和删除已录入的记录

**操作流程**：
1. 点击"计件录入"按钮
2. 选择仓库和品类
3. 输入件数和相关费用
4. 提交保存

---

### 2. 考勤打卡 ⏰
**图标**：橙色时钟打勾图标  
**功能**：快速进行考勤打卡  
**跳转页面**：`/pages/driver/clock-in/index`

**使用场景**：
- 每天上班时打卡签到
- 记录出勤时间
- 查看打卡历史记录

**操作流程**：
1. 点击"考勤打卡"按钮
2. 选择要打卡的仓库
3. 点击打卡按钮
4. 系统记录打卡时间

**注意事项**：
- 迟到会被系统自动标记
- 每天只能打卡一次
- 打卡时间由系统自动记录

---

### 3. 请假申请 📅
**图标**：紫色日历删除图标  
**功能**：快速提交请假申请  
**跳转页面**：`/pages/driver/leave/index`

**使用场景**：
- 需要请假时快速提交申请
- 查看请假申请状态
- 管理请假草稿

**操作流程**：
1. 点击"请假申请"按钮
2. 选择请假类型（快捷请假/补请假/离职申请）
3. 填写请假信息
4. 提交申请等待审批

**请假类型**：
- **快捷请假**：通过滚轮快速选择请假天数
- **补请假**：补录历史请假记录
- **离职申请**：提交离职申请（需提前指定天数）

**功能特点**：
- 智能天数计算
- 月度请假统计
- 草稿保存功能
- 实时审批状态

---

### 4. 数据统计 📊
**图标**：绿色柱状图图标  
**功能**：查看个人数据统计  
**跳转页面**：`/pages/driver/piece-work/index`

**使用场景**：
- 查看个人计件记录
- 统计收入情况
- 分析工作数据

**操作流程**：
1. 点击"数据统计"按钮
2. 查看计件记录列表
3. 查看收入统计
4. 筛选和分析数据

**统计内容**：
- 计件记录明细
- 收入汇总
- 按日期筛选
- 按仓库筛选

---

## 设计特点

### 1. 视觉设计
- **渐变背景**：每个按钮使用不同颜色的渐变背景
  - 计件录入：蓝色渐变（from-blue-50 to-blue-100）
  - 考勤打卡：橙色渐变（from-orange-50 to-orange-100）
  - 请假申请：紫色渐变（from-purple-50 to-purple-100）
  - 数据统计：绿色渐变（from-green-50 to-green-100）

- **图标设计**：使用Material Design Icons
  - 图标尺寸：text-5xl（大图标，易识别）
  - 图标颜色：与背景渐变色系一致

- **文字设计**：
  - 字体大小：text-base（适中，易读）
  - 字体粗细：font-medium（中等粗细）
  - 字体颜色：text-gray-800（深灰色，对比度好）

### 2. 交互设计
- **点击反馈**：active:scale-95（点击时缩小5%）
- **过渡动画**：transition-all（平滑过渡）
- **圆角设计**：rounded-xl（大圆角，现代感）
- **内边距**：p-6（充足的内边距，易点击）

### 3. 布局设计
- **网格布局**：grid grid-cols-2（2列布局）
- **间距**：gap-4（按钮间距适中）
- **对齐方式**：flex flex-col items-center（垂直居中对齐）
