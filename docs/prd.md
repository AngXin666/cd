# 司机管理后台电脑端需求文档

## 1. 功能概述
### 1.1 系统名称
司机管理后台电脑端

### 1.2 系统描述
专为老板和车队长打造的电脑端管理后台系统，提供完整的司机管理和车队运营功能，支持便捷的网页访问和管理操作。

## 2. 核心功能
### 2.1 系统登录功能
- **账号登录**：支持邮箱或手机号登录
- **用户认证**：老板和车队长使用各自账号密码登录
- **权限验证**：根据用户角色分配不同功能权限
- **会话管理**：保持登录状态，自动刷新token
- **安全策略**：密码强度要求，登录失败次数限制

### 2.2 首页展示功能
- **欢迎界面**：显示用户姓名的欢迎语
- **基本信息**：展示用户角色（老板/车队长）和联系方式
- **快捷入口**：提供常用功能的快速访问链接
- **系统公告**：展示重要通知和提醒信息
- **详细统计**：提供全面的工作统计数据和图表展示

### 2.3 司机管理功能
- **司机信息管理**：添加、编辑、删除司机资料
- **出勤记录**：查看和管理司机出勤情况
- **任务分配**：为司机分配工作任务
- **绩效考核**：查看司机工作表现和考核结果
- **通讯录**：管理司机联系方式和紧急联系人

### 2.4 车队管理功能
- **车辆信息**：管理车队车辆资料
- **保养记录**：记录车辆保养和维修信息
- **油耗统计**：统计车辆油耗和费用
- **保险管理**：跟踪车辆保险有效期

### 2.5 数据统计功能
- **出勤统计**：按时间段统计司机出勤率
- **任务完成率**：展示任务执行情况
- **费用统计**：统计车队运营成本
- **效率分析**：分析司机和车队工作效率

## 3. 系统逻辑流程
### 3.1 系统登录流程
1. 用户访问登录页面
2. 输入账号和密码
3. 系统验证账号有效性
4. 验证密码正确性
5. 检查用户角色（老板/车队长）
6. 生成访问token
7. 跳转到对应首页

### 3.2 首页访问流程
1. 用户登录成功后自动跳转到管理首页
2. 系统根据用户角色显示个性化内容
3. 支持快速访问常用功能模块
4. 显示最新的系统通知和提醒
5. 展示详细的业务数据统计图表

### 3.3 司机管理流程
1. 系统自动同步司机数据
2. 支持批量导入司机信息
3. 提供完整的司机档案管理
4. 实时更新出勤和任务记录

### 3.4 数据统计流程
1. 系统自动收集各项业务数据
2. 按预设维度进行数据统计
3. 生成可视化的图表展示
4. 支持数据导出和打印

## 4. 用户界面布局
### 4.1 登录页面布局
- **账号输入框**：支持邮箱或手机号登录
- **密码输入框**：带密码强度提示
- **登录按钮**：提交登录信息
- **忘记密码**：提供密码重置功能
- **角色标识**：显示用户角色（老板/车队长）

### 4.2 首页界面布局
- **顶部导航栏**：包含用户信息和基础菜单
- **欢迎区域**：显示用户姓名和角色的欢迎语
- **信息卡片**：展示联系方式等基本信息
- **快捷入口区**：采用卡片式布局展示功能入口
- **公告区域**：显示系统通知和提醒
- **统计数据**：详细的业务数据图表展示

### 4.3 司机管理界面
- **司机列表**：显示所有司机的基本信息
- **筛选功能**：按角色、状态等条件筛选
- **快速操作**：支持编辑、删除等批量操作
- **详情页面**：查看司机完整档案和记录\n
### 4.4 数据统计界面
- **仪表盘**：展示关键业务指标
- **图表区域**：柱状图、折线图等多种图表展示
- **筛选工具**：按时间、维度等条件筛选数据
- **导出功能**：支持Excel和PDF格式导出

## 6. 设计风格
### 6.1 配色方案
- 主色调：蓝色系渐变（#1890ff到#4a90e2），体现专业可靠
- 强调色：橙色（#ff7a45）用于重要操作按钮
- 成功提示：绿色（#52c41a），错误提示：红色（#ff4d4f）
- 背景色：浅灰色（#f5f5f5）作为主要背景色

### 6.2 视觉细节
- 圆角设计：统一使用4px圆角，营造现代商务感
- 卡片阴影：轻微阴影效果增强层次感
- 图标风格：线性图标配合填充图标，保持视觉一致性
- 按钮样式：主要按钮采用填充样式，次要按钮采用边框样式

### 6.3 整体布局
- 采用卡片式布局，信息层次清晰
- 响应式设计，适配不同电脑屏幕分辨率
- 简洁的顶部导航，便于快速访问
- 数据展示采用清晰的表格和图表形式
- 左侧导航布局，便于功能分类管理
