# 司机管理后台简化版需求文档

## 1. 功能概述
### 1.1 系统名称
司机管理后台简化版

### 1.2 系统描述
为老板和车队长提供便捷登录和基础管理的电脑端后台系统，支持小程序用户通过账号密码登录访问管理首页，并提供小程序管理功能。

## 2. 核心功能
### 2.1 系统登录功能
- **账号登录**：支持邮箱或手机号登录
- **用户认证**：老板和车队长使用各自账号密码登录
- **权限验证**：根据用户角色分配不同功能权限
- **会话管理**：保持登录状态，自动刷新token
- **安全策略**：密码强度要求，登录失败次数限制

### 2.2 首页展示功能
- **欢迎界面**：显示用户姓名的欢迎语
- **基本信息**：展示用户角色（老板/车队长）和联系方式
- **快捷入口**：提供常用功能的快速访问链接
- **系统公告**：展示重要通知和提醒信息
- **基础统计**：显示简单的工作统计数据

### 2.3 小程序管理功能
- **用户数据同步**：与小程序用户数据实时同步
- **小程序配置管理**：管理小程序的基础设置和参数
- **版本管理**：查看小程序版本信息和更新记录
- **数据统计**：查看小程序使用统计数据
- **消息推送**：向小程序用户发送通知消息

## 3. 系统逻辑流程
### 3.1 系统登录流程
1. 用户访问登录页面
2. 输入账号和密码
3. 系统验证账号有效性
4. 验证密码正确性
5. 检查用户角色（老板/车队长）
6. 生成访问token
7. 跳转到对应首页

### 3.2 首页访问流程
1. 用户登录成功后自动跳转到管理首页
2. 系统根据用户角色显示个性化内容
3. 支持快速访问常用功能模块
4. 显示最新的系统通知和提醒

### 3.3 小程序管理流程
1. 系统自动检测小程序连接状态
2. 实时同步小程序用户数据
3. 提供小程序配置和管理界面
4. 支持数据统计查看和消息推送

## 4. 用户界面布局
### 4.1 登录页面布局
- **账号输入框**：支持邮箱或手机号登录
- **密码输入框**：带密码强度提示
- **登录按钮**：提交登录信息
- **忘记密码**：提供密码重置功能
- **角色标识**：显示用户角色（老板/车队长）

### 4.2 首页界面布局
- **顶部导航栏**：包含用户信息和基础菜单
- **欢迎区域**：显示用户姓名和角色的欢迎语
- **信息卡片**：展示联系方式等基本信息
- **快捷入口区**：采用卡片式布局展示功能入口
- **公告区域**：显示系统通知和提醒
- **统计数据**：简单的业务数据展示

### 4.3 小程序管理界面
- **小程序状态**：显示连接状态和版本信息
- **用户管理**：查看和管理小程序用户数据
- **配置管理**：小程序基础设置和参数配置
- **数据统计**：小程序使用情况统计
- **消息中心**：消息推送管理界面

## 6. 设计风格
### 6.1 配色方案\n- 主色调：蓝色系渐变（#1890ff到#4a90e2），体现专业可靠
- 强调色：橙色（#ff7a45）用于重要操作按钮
- 成功提示：绿色（#52c41a），错误提示：红色（#ff4d4f）
- 背景色：浅灰色（#f5f5f5）作为主要背景色

### 6.2 视觉细节
- 圆角设计：统一使用4px圆角，营造现代商务感
- 卡片阴影：轻微阴影效果增强层次感
- 图标风格：线性图标配合填充图标，保持视觉一致性
- 按钮样式：主要按钮采用填充样式，次要按钮采用边框样式

### 6.3 整体布局
- 采用卡片式布局，信息层次清晰
- 响应式设计，适配不同电脑屏幕分辨率
- 简洁的顶部导航，便于快速访问
- 数据展示采用清晰的表格和图表形式
- 小程序管理界面采用左侧导航布局
