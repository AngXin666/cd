# 上楼费和分拣费显示优化

## 修改概述

在所有用户端品类统计的地方添加了上楼费和分拣费的显示，确保费用信息完整透明。

## 修改的页面

### 1. 数据汇总页面 (src/pages/manager/data-summary/index.tsx)

**修改内容：**
- 在记录列表中添加了分拣费的显示
- 在品类标签旁边添加了"需分拣"标识（绿色）
- 修改了"需上楼"标识的颜色（从蓝色改为橙色）

**显示效果：**
- 基础费用：数量 × 单价（灰色）
- 上楼费：单价（橙色）
- 分拣费：件数 × 单价（绿色）
- 总金额：包含所有费用（绿色加粗）

### 2. 普通管理员计件管理页面 (src/pages/manager/piece-work/index.tsx)

**修改内容：**
- 在记录列表中添加了"需分拣"标识（绿色）
- 在费用信息中添加了分拣费的显示
- 重新组织了费用信息的布局，使其更清晰

**显示效果：**
- 标识：需上楼（橙色）、需分拣（绿色）
- 费用明细：数量、单价、上楼费、分拣费
- 总金额：包含所有费用

### 3. 超级管理员计件管理页面 (src/pages/super-admin/piece-work/index.tsx)

**修改内容：**
- 在记录列表的费用信息中添加了分拣费的显示

**显示效果：**
- 与普通管理员页面保持一致

## 费用计算说明

### total_amount 的计算公式

```typescript
total_amount = 基础费用 + 上楼费 + 分拣费
```

其中：
- **基础费用** = 数量 × 单价
- **上楼费** = 数量 × 上楼单价（如果需要上楼）
- **分拣费** = 分拣件数 × 分拣单价（如果需要分拣）

### 计算位置

所有表单和编辑功能都正确计算了 total_amount：
1. 司机录入页面 (src/pages/driver/piece-work-entry/index.tsx)
2. 司机编辑功能 (src/pages/driver/piece-work/index.tsx)
3. 普通管理员表单 (src/pages/manager/piece-work-form/index.tsx)
4. 超级管理员表单 (src/pages/super-admin/piece-work-form/index.tsx)

## 颜色规范

为了更好地区分不同类型的费用，使用了以下颜色规范：

- **基础费用**：灰色 (text-gray-600)
- **上楼费**：橙色 (text-orange-600)
- **分拣费**：绿色 (text-green-600)
- **总金额**：深蓝色/绿色 (text-blue-900 / text-green-600)

## 统计数据准确性

所有统计功能都使用 `record.total_amount` 进行汇总，因此统计数据已经包含了：
- 基础费用
- 上楼费
- 分拣费

不需要额外修改统计逻辑。

## 用户体验改进

1. **信息完整性**：用户可以清楚地看到每笔费用的构成
2. **视觉区分**：使用不同颜色区分不同类型的费用
3. **标识清晰**：使用标签标识"需上楼"和"需分拣"
4. **布局优化**：费用信息排列更加清晰，易于阅读
