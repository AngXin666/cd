# 个人中心功能使用说明

## 功能概述

个人中心模块为车队管家小程序提供了完整的个人信息管理、账户安全和用户反馈功能。所有用户（司机、管理员、超级管理员）都可以使用这些功能。

---

## 功能列表

### 1. 个人中心主页

**访问路径**：底部导航栏 → 我的

**功能说明**：
- 显示用户头像（支持自定义上传）
- 显示姓名、昵称和角色标签
- 展示个人基本信息：
  - 手机号（中间4位自动隐藏保护隐私）
  - 邮箱地址
  - 居住地（省份 + 城市）
  - 紧急联系人信息
- 提供快捷入口：
  - 编辑资料
  - 设置
  - 帮助与反馈
- 退出登录功能

---

### 2. 编辑资料

**访问路径**：我的 → 编辑资料

#### 2.1 头像上传
- 点击头像或"上传头像"按钮
- 支持从相册选择或拍照
- 自动压缩至1MB以内
- 支持格式：JPG、PNG、WEBP

#### 2.2 基本信息
- **姓名**（必填）：真实姓名，用于系统显示
- **昵称**（选填）：个性化昵称
- **邮箱**（选填）：用于接收通知，需符合邮箱格式
- **手机号**：只读显示，不可修改

#### 2.3 居住地址
- **省份**：下拉选择器，包含全国34个省级行政区
- **城市**：手动输入
- **区县**：手动输入
- **详细地址**：最多100字

#### 2.4 紧急联系人
- **姓名**：紧急联系人姓名
- **电话**：11位手机号，需符合格式

**保存规则**：
- 姓名为必填项
- 邮箱需符合格式（如：example@domain.com）
- 紧急联系人电话需符合11位手机号格式
- 保存成功后自动返回个人中心

### 4. 修改密码

**访问路径**：我的 → 设置 → 修改密码

#### 4.1 密码要求
- 长度至少8位
- 必须包含字母和数字
- 建议使用大小写字母、数字和符号组合

#### 4.2 修改流程
1. 输入原密码
2. 输入新密码（实时显示密码强度）
3. 确认新密码（实时验证是否匹配）
4. 点击"确认修改"

#### 4.3 密码强度指示器
- **红色**：密码不符合要求（长度不足或缺少字母/数字）
- **绿色**：密码强度强

#### 4.4 注意事项
- 新密码不能与原密码相同
- 两次输入的新密码必须一致
- 修改成功后需要重新登录

### 6. 意见反馈

**访问路径**：我的 → 帮助与反馈 → 意见反馈

#### 6.1 提交反馈

**反馈类型**：
- 功能建议
- 问题反馈
- 功能需求
- 投诉建议
- 其他

**反馈内容**：
- 至少10个字
- 最多500字
- 详细描述您的问题或建议

**联系方式**（选填）：
- 方便我们回复您的反馈
- 可以是手机号、邮箱或其他联系方式

**提交流程**：
1. 选择反馈类型
2. 输入反馈内容（至少10个字）
3. 填写联系方式（可选）
4. 点击"提交反馈"
5. 提交成功后自动切换到历史反馈

#### 6.2 历史反馈

**查看方式**：
- 点击"历史反馈"标签
- 显示所有已提交的反馈记录

**反馈状态**：
- **待处理**（黄色标签）：反馈已提交，等待处理
- **处理中**（蓝色标签）：管理员正在处理
- **已解决**（绿色标签）：问题已解决

**显示信息**：
- 反馈类型
- 反馈内容
- 联系方式（如果填写）
- 提交时间
- 当前状态

## 使用技巧

### 1. 头像上传
- 建议使用清晰的正面照片
- 照片大小不超过1MB
- 如果照片过大，系统会自动压缩

### 2. 密码安全
- 定期修改密码，提高账户安全性
- 不要使用过于简单的密码
- 不要与其他平台使用相同密码

### 3. 个人信息
- 及时更新个人信息，方便管理员联系
- 紧急联系人信息很重要，请认真填写
- 居住地信息可以帮助管理员更好地安排工作

### 4. 意见反馈
- 详细描述问题，有助于快速解决
- 提供联系方式，方便我们回复
- 定期查看反馈状态，了解处理进度

## 常见问题

### Q1: 如何上传头像？
A: 进入"我的" → "编辑资料"，点击头像或"上传头像"按钮，选择照片即可。

### Q2: 手机号可以修改吗？
A: 手机号是账户的唯一标识，不支持修改。如需更换，请联系管理员。

### Q3: 忘记密码怎么办？
A: 请联系管理员重置密码。登录后可以在"设置"中修改密码。

### Q4: 提交的反馈多久会得到回复？
A: 我们会在1-3个工作日内处理您的反馈，并更新反馈状态。

### Q5: 如何查看反馈处理进度？
A: 进入"意见反馈"页面，点击"历史反馈"标签，查看反馈状态。
