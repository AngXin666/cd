## 📊 核心功能

### 1. 数据统计
- 请假申请统计（总数、待审批、已通过、已驳回）
- 离职申请统计（总数、待审批、已通过、已驳回）

### 2. 快速搜索
- 支持搜索司机姓名、仓库名称、事由
- 实时搜索，输入即过滤

### 3. 基础筛选
- 按审批状态筛选（全部/待审批/已通过/已驳回）

### 4. 高级筛选
- 请假类型筛选（病假/事假/年假/其他）
- 所属仓库筛选
- 当事人（司机）筛选
- 审批人筛选（全部/未审批/具体审批人）
- 时间范围筛选（开始日期-结束日期）

### 5. 申请展示
- 清晰的卡片式布局
- 突出的当事人信息
- 详细的请假/离职信息
- 完整的审批历史
- 明确的操作按钮

### 6. 审批操作
- 通过申请
- 驳回申请
- 自动记录审批人和时间

## 📈 使用场景

### 场景1：日常审批
超级管理员每天登录系统，查看待审批申请，逐一审批。

## 📝 快速开始

### 1. 访问功能
进入超级管理员后台 → 点击"请假审批"

### 2. 查看待审批申请
点击"待审批"状态按钮 → 查看所有待审批的申请

### 3. 审批申请
找到需要审批的申请 → 点击"通过"或"驳回" → 确认操作

### 4. 搜索申请
在搜索框输入关键字 → 查看搜索结果

### 5. 筛选申请
展开高级筛选 → 设置筛选条件 → 查看筛选结果

## 🔧 常见问题

### Q：如何快速找到某个司机的申请？
**A：** 在搜索框直接输入司机姓名，或展开高级筛选选择该司机。

### Q：如何查看未审批的申请？
**A：** 点击"待审批"状态按钮。

### Q：如何同时按多个条件筛选？
**A：** 展开高级筛选，同时设置多个筛选条件。

### Q：审批后可以撤销吗？
**A：** 当前版本不支持撤销，请谨慎操作。

### Q：如何重置筛选条件？
**A：** 展开高级筛选，点击"清空所有筛选条件"按钮。
