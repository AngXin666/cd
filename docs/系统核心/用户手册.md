## 系统简介

车队管家是一款专为车队管理打造的微信小程序，提供多角色权限管理，满足车队运营的分层管理需求。

### 系统特点

- **多角色权限管理**：支持系统管理员、老板、车队长、司机四种角色
- **物理隔离架构**：每个租户拥有独立的数据库，数据安全有保障
- **实时通知系统**：重要事件及时通知相关人员
- **移动优先设计**：专为微信小程序优化，操作简单便捷

### 角色说明

| 角色 | 权限范围 | 主要功能 |
|------|---------|---------|
| 系统管理员 | 管理所有租户 | 创建租户、管理租户信息 |
| 老板 | 管理租户内所有数据 | 用户管理、车辆管理、数据统计 |
| 平级账号 | 根据权限类型决定 | 与老板相同或只读权限 |
| 车队长 | 管理指定范围的数据 | 考勤管理、请假审批、计件管理 |
| 司机 | 只能查看和管理自己的数据 | 打卡、请假、计件记录 |

## 系统管理员

系统管理员负责管理所有租户，包括创建租户、查看租户信息等。

### 租户管理

#### 查看租户列表
1. 登录系统管理员账号
2. 进入"租户列表"页面
3. 查看所有租户信息

#### 创建租户
1. 点击"创建租户"按钮
2. 填写租户信息：
   - 公司名称
   - 联系人姓名
   - 联系人电话
   - 联系人邮箱
   - 老板姓名
   - 老板电话
   - 老板邮箱
   - 老板密码
3. 点击"提交"按钮
4. 等待系统创建租户
5. 创建成功后，系统会自动创建租户数据库和老板账号

#### 编辑租户信息
1. 在租户列表中找到要编辑的租户
2. 点击"编辑"按钮
3. 修改租户信息
4. 点击"保存"按钮

#### 删除租户
1. 在租户列表中找到要删除的租户
2. 点击"删除"按钮
3. 确认删除操作
4. 系统会删除租户数据库和所有相关数据

## 通知系统

### 查看通知

#### 通知列表
1. 点击顶部的通知图标
2. 查看所有通知
3. 未读通知会显示红点

#### 通知详情
1. 在通知列表中点击通知

## 常见问题

### 登录问题

**Q: 忘记密码怎么办？**
A: 点击登录页面的"忘记密码"链接，通过手机验证码重置密码。

**Q: 登录后跳转到错误的页面？**
A: 请联系管理员检查您的账号角色设置。

### 权限问题

**Q: 无法查看某些数据？**
A: 请检查您的账号角色和权限设置，联系管理员调整权限。

**Q: 无法执行某些操作？**
A: 请确认您的账号状态是否为"活跃"，以及是否有相应的操作权限。

### 数据问题

**Q: 数据没有及时更新？**
A: 请尝试下拉刷新页面，或者退出重新登录。

**Q: 数据丢失了？**
A: 请联系管理员检查数据库，可能是权限问题导致无法查看。

### 通知问题

**Q: 收不到通知？**
A: 请检查通知设置是否开启，以及微信小程序的通知权限是否允许。

**Q: 通知太多怎么办？**
A: 可以在通知设置中关闭不需要的通知类型。

### 其他问题

**Q: 小程序卡顿或闪退？**
A: 请尝试清理微信缓存，或者重启微信。

**Q: 遇到其他问题怎么办？**
A: 请联系系统管理员或技术支持人员。
