# 超级管理员查看司机信息字段排序优化

## 修改位置
- **页面路径**：`src/pages/super-admin/user-detail/index.tsx`
- **功能入口**：超级管理员端 → 车辆管理 → 点击"查看司机"按钮

## 优化内容

### 1. 身份证信息部分

**调整后的显示顺序**：

1. ✅ **姓名** - 真实姓名（从第2位调整到第1位）
2. ✅ **电话号码** - 联系方式（新增字段）
3. ✅ **出生日期** - 出生日期（从第3位保持不变）
4. ✅ **身份证号码** - 18位身份证号（从第1位调整到第4位）
5. ✅ **身份证地址** - 户籍地址（从第4位调整到第5位）
6. ✅ **签发机关** - 身份证签发机关（从第5位调整到第6位）

**优化说明**：
- 将最重要的**姓名**信息提到第一位，方便快速识别司机身份
- 新增**电话号码**字段，显示司机的联系方式（从 `userInfo.phone` 获取）
- 将身份证号码后移，因为姓名和电话更常用于日常管理

### 2. 驾驶证信息部分

**调整后的显示顺序**：

1. ✅ **驾驶证号码** - 驾驶证号码（保持第1位）
2. ✅ **准驾车型** - 如C1、B2等（保持第2位）
3. ✅ **初次领证日期** - 首次领取驾驶证的日期（从第4位调整到第3位）
4. ✅ **有效期限** - 驾驶证有效期范围（从第3位调整到第4位）

**优化说明**：
- 将**初次领证日期**和**有效期限**的位置对换
- 初次领证日期是固定不变的历史信息，更适合放在前面
- 有效期限是需要关注的时效性信息，放在后面更符合阅读习惯

## 技术实现

### 电话号码字段实现

```tsx
{/* 电话号码 */}
{userInfo.phone && (
  <View className="bg-gradient-to-r from-sky-50 to-sky-100 rounded-xl p-4 mb-3">
    <View className="flex items-center justify-between">
      <View className="flex-1">
        <Text className="text-xs text-sky-600 mb-1 block">电话号码</Text>
        <Text className="text-base text-sky-900 font-bold block tracking-wider">
          {userInfo.phone}
        </Text>
      </View>
      <View className="bg-sky-200 rounded-full w-10 h-10 flex items-center justify-center">
        <View className="i-mdi-phone text-xl text-sky-700"></View>
      </View>
    </View>
  </View>
)}
```

**数据来源**：
- `userInfo.phone` - 从 `profiles` 表中获取的司机电话号码
- 如果司机没有填写电话号码，该字段不显示
