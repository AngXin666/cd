# 自动登录功能使用指南

## 🎉 功能介绍

车队管家小程序现已支持自动登录功能！您只需登录一次，下次打开应用时会自动登录，无需重复输入账号密码。

## 📱 如何使用

### 首次登录

1. **打开应用**
   - 打开车队管家小程序
   - 看到登录页面

2. **输入账号密码**
   - 输入您的手机号或邮箱
   - 输入密码

3. **点击登录**
   - 点击"登录"按钮
   - 等待登录成功

4. **登录成功**
   - 自动跳转到主界面
   - 开始使用应用

### 自动登录

1. **关闭应用**
   - 正常关闭小程序
   - 或切换到其他应用

2. **重新打开**
   - 再次打开车队管家小程序
   - **自动登录，无需输入账号密码**
   - 直接进入主界面

3. **开始使用**
   - 无需等待
   - 立即使用应用功能

### 退出登录

1. **进入个人中心**
   - 点击底部"我的"标签
   - 进入个人中心页面

2. **点击退出登录**
   - 找到"退出登录"按钮
   - 点击按钮

3. **确认退出**
   - 在弹出的确认对话框中
   - 点击"确定"

4. **退出成功**
   - 自动跳转到登录页面
   - 下次打开需要重新登录

## 💡 常见问题

### Q1：自动登录安全吗？
**A：** 非常安全！
- 登录信息加密存储在您的设备上
- 其他人无法访问
- 定期自动刷新，确保安全性

### Q2：自动登录会保持多久？
**A：** 长期有效！
- 只要不主动退出登录
- 系统会自动保持登录状态
- 无需担心过期问题

### Q3：换手机后还能自动登录吗？
**A：** 需要重新登录。
- 登录信息保存在设备上
- 换设备后需要重新登录
- 登录一次后，新设备也会自动登录

### Q4：如何关闭自动登录？
**A：** 点击退出登录即可。
- 进入个人中心
- 点击"退出登录"
- 下次打开需要重新登录

### Q5：忘记密码怎么办？
**A：** 可以重置密码。
- 在登录页面点击"忘记密码"
- 按照提示重置密码
- 使用新密码登录

### Q6：多个账号可以自动登录吗？
**A：** 一次只能保持一个账号登录。
- 切换账号需要先退出当前账号
- 然后登录新账号
- 新账号会自动保持登录状态

### Q7：自动登录失败怎么办？
**A：** 可能的原因和解决方法：
- **网络问题**：检查网络连接
- **账号异常**：联系管理员
- **版本过旧**：更新到最新版本
- **清除缓存**：重新登录

### Q8：卸载应用后重新安装，还能自动登录吗？
**A：** 不能。
- 卸载应用会清除所有数据
- 重新安装后需要重新登录
- 登录后会恢复自动登录功能
