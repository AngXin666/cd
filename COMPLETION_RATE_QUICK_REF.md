# 完成率状态快速参考

## 状态分类

| 状态 | 完成率范围 | 颜色 | 图标 | 说明 |
|------|-----------|------|------|------|
| 超额完成 | > 110% | 🟢 绿色 | 🏆 奖杯 | 表现优秀，超出预期 |
| 达标 | 100% - 110% | 🔵 蓝色 | ✓ 对勾 | 正常完成，符合要求 |
| 不达标 | 70% - 99% | 🟠 橙色 | ⚠ 警告 | 需要关注，接近目标 |
| 严重不达标 | < 70% | 🔴 红色 | ⛔ 严重警告 | 需要立即关注 |

## 颜色代码

### 超额完成（绿色系）
- 徽章背景：`linear-gradient(135deg, #10b981, #059669)`
- 状态卡片背景：`#dcfce7`
- 状态卡片文字：`#15803d`
- 圆环颜色：`#10b981`

### 达标（蓝色系）
- 徽章背景：`linear-gradient(135deg, #3b82f6, #2563eb)`
- 状态卡片背景：`#dbeafe`
- 状态卡片文字：`#1e40af`
- 圆环颜色：`#3b82f6`

### 不达标（橙色系）
- 徽章背景：`linear-gradient(135deg, #f97316, #ea580c)`
- 状态卡片背景：`#fed7aa`
- 状态卡片文字：`#c2410c`
- 圆环颜色：`#f97316`

### 严重不达标（红色系）
- 徽章背景：`linear-gradient(135deg, #ef4444, #dc2626)`
- 状态卡片背景：`#fecaca`
- 状态卡片文字：`#b91c1c`
- 圆环颜色：`#ef4444`

## 图标代码

- 超额完成：`i-mdi-trophy`
- 达标：`i-mdi-check-circle`
- 不达标：`i-mdi-alert-circle`
- 严重不达标：`i-mdi-alert-octagon`

## 使用方法

```typescript
// 1. 获取状态
const status = getCompletionRateStatus(summary.completionRate || 0)

// 2. 使用状态属性
status.label        // "超额完成" | "达标" | "不达标" | "严重不达标"
status.bgColor      // 状态卡片背景色
status.textColor    // 状态卡片文字颜色
status.ringColor    // 圆环颜色
status.badgeBgColor // 徽章背景色（渐变）
```
