# 登录测试指南

## 🎯 快速测试

### 测试账号1：超级管理员
```
账号：admin
密码：123456
```
或
```
账号：13800000001
密码：123456
```

### 测试账号2：普通管理员
```
账号：admin1
密码：123456
```
或
```
账号：13800000002
密码：123456
```

### 测试账号3：司机
```
账号：admin2
密码：123456
```
或
```
账号：13800000003
密码：123456
```

## ✅ 测试步骤

1. 打开登录页面
2. 选择"密码登录"方式
3. 输入上述任一账号和密码
4. 点击"密码登录"按钮
5. 应该能够成功登录并跳转到对应的工作台

## 🔍 预期结果

- **超级管理员**：可以访问所有功能，包括用户管理、仓库管理等
- **普通管理员**：可以管理北京仓库的数据
- **司机**：可以查看自己的考勤、计件等数据

## ❌ 如果登录失败

请检查：
1. 账号和密码是否输入正确（注意大小写）
2. 密码是否为 `123456`（6位数字）
3. 网络连接是否正常
4. 浏览器控制台是否有错误信息

## 📞 技术支持

如果问题仍然存在，请提供以下信息：
- 使用的账号
- 错误提示信息
- 浏览器控制台的错误日志
